Claude.md Topic Analyzer & Visualizer

How to use:

  1. Ensure your GitHub Personal Access Token (PAT) is set as an environment variable named `GITHUB_PAT` on Render.com.
  2. Click "Start Analysis". This will fetch `claude.md` files from public GitHub repos, preprocess them, train an LDA model, and generate an interactive topic visualization. This might take a few minutes.
  3. Once the analysis is complete (a success message will appear), click "View Visualization" to open the interactive `pyLDAvis` chart in a new tab.
  4. The `pyLDAvis` chart allows you to explore the discovered topics and their most relevant words.

Note: The GitHub API has rate limits. If you hit limits, the collection might be incomplete. The MVP currently limits collection to 500 files to manage this.
